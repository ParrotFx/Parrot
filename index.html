Parrot

All new View Engine for Asp.Net MVC

View project onGitHub

Model

return View(new {
  Header = "Parrot",
  Features = new [] {
    "Familiar syntax",
    "Extensible output",
    "Simple model binding"
  }
});

Parrot Template

doctype
html {
  head {
    title("This is Parrot!")
  }
  body {
    h1#header(Header)
    p#tagname.small("All new Asp.Net View Engine")
    div#description.rounded-corners {
      h2("Features")
      ul(Features) {
        li(this)
      }	  
    }
  }
}
