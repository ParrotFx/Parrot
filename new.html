What is Parrot?

Parrot is a templating engine that has been simplified for quick and easy use.

Simplicity

Parrot was designed to be simple by using the familiar CSS selector syntax.

Flexibility

Easily extend and replace renderers to output custom html.

Model support

The default html renderers that ship with Parrot.Mvc have great model support. Automatically scoped variables as well as support for IEnumerables.

Install

Run the following command in the Package Manager Console

PM> Install-Package Parrot.Mvc

Register with ASP.NET MVC

protected void Application_Start() { ViewEngines.Engines.Add(new ParrotViewEngine()); }

Sample call

var model = new { Header = "Parrot", Tag = "All new Asp.Net View Engine", Features = new [] { "Familiar syntax", "Extensible output", "Simple model binding" }; return View(model);
